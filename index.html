This portfolio includes my work in Mrs. Ashish's Web Design class and my reflection for each unit.

In Unit Two, we learned the fundamentals of HTML (Hypertext Markup Language). HTML is the language that supplies the information available on a web page. By learning the principles of HTML, we are laying the foundation to be able to create more complex web pages in order to one day publish as a completed website.

In addition to HTML, we also learned how to apply CSS (Cascading Style Sheets). CSS is the language that alters the appearance of the content on a web page. By using CSS, we are able to change elements including: font size, background color, font families, and margins. Additionally, we can use CSS to alter the positions of elements on the page, as well as change their height and width.

In Unit 4, we were introduced to the basics of using web graphics on our site. We learned about copyright restrictions on the web, how to give credit to the owner/originator of the image, and the different types of copyright that can be applied to an online image. In addition, we learned how to create a photo album on our site by resizing our images. We also used Photoshop and/or Microsoft Paint 3D to design and implement a button and web page banner onto our website.

UNIT FIVE

In Unit 5, we were introduced to JavaScript, using it to design some basic functions. We learned how to write the code into the head portion of our page, using the script tag. We also learned how to write variables in order to assign values to pieces of code. Using these variables, we were able to create a simple button using the showAlert function, as well as a clock that we can put on our home screen. Overall, JavaScript will be a useful tool for adding more detailed functionality into our web sites in the future.

UNIT SIX

In Unit 6, we developed our skills in understanding how to better create a web page by studying and analyzing and testing the validation and usability of our sites and others. We learned how to use online HTML validation resources to identify potential errors in the code of our website, to use to improve the construction. In addition, we as a class formed groups in which we tested each others' usability, identifying potential design errors in our pages. These could include graphics, text size and color, and the number of links and buttons, all elements that can be improved to create a superior site.

UNIT SEVEN

In Unit 7, we learned the different ways to use web authoring software to create a page to be published on the web. We learned the difference between web authoring software and content management systems, or CMS, and used this information to improve our own web sites. In addition, we learned how to publish our page to the web. To do this, we used GitHub, an online publishing resource, and published our personal projects to the web for everyone to see.
